# Projetos Java — Portfólio de Aprendizado
Repositório com projetos progressivos em Java, do básico ao avançado, seguindo um plano de estudos.

## Sobre

Este repositório reúne projetos desenvolvidos em Java ao longo de uma trilha de estudos estruturada. O objetivo é consolidar fundamentos, evoluir para práticas modernas e construir aplicações backend seguindo padrões utilizados no mercado.

## Objetivos

* Evoluir do nível iniciante ao nível júnior em Java
* Aplicar boas práticas de desenvolvimento e organização de código
* Desenvolver aplicações progressivamente mais completas
* Construir um portfólio técnico consistente

## Estrutura do repositório

```
projetos-java/
 ├── mes-1-sistema-cadastro/
 ├── mes-2-sistema-arquivo/
 ├── mes-3-refatoracao/
 ├── mes-4-api-spring/
 ├── mes-5-api-seguranca/
 └── projeto-final/
```

## Conteúdo por etapa

**Mês 1 — Fundamentos**
Lógica de programação, estruturas de controle, métodos e listas.
Projeto: sistema de cadastro em console (CRUD).

**Mês 2 — Intermediário (POO)**
Programação orientada a objetos, collections, exceções e persistência em arquivos.
Projeto: sistema com armazenamento em arquivo.

**Mês 3 — Java moderno e boas práticas**
Stream API, expressões lambda, Optional, refatoração e testes com JUnit.

**Mês 4 — Backend com Spring Boot**
Construção de APIs REST, organização em camadas e integração com banco de dados.

**Mês 5 — Segurança e padrões**
Autenticação, autorização, JWT e boas práticas de API.

**Mês 6 — Projeto final**
Desenvolvimento de uma aplicação completa, com foco em organização, qualidade e padrão profissional.
